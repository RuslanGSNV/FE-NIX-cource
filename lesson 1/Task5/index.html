Содержание

  1. Заголовки (h1, h2, h3, h4, h5, h6);
  2. Абазацы (p);
  3. Списки (ul, ol, dl);
  4. Блоки (div);




1.Заголовки

Для определения заголовков исполюзуются теги шести уровноей - h1, h2, h3, h4, h5, h6.
Заголовки имеют следующий вид:

Заголовок 1

Заголовок 2

Заголовок 3

Заголовок 4

Заголовок 5
Заголовок 6





2.Абазацы

Для определения абзаца используется тег p.
Любой текст, заключенный между открывающим и закрывающим тегами p будут считаться абзацом.
Тег p является блочным тегом, поэтому будет занимать всю ширину страницы либо родительского блока,
высота же будет зависить от контента.





3.Списки

Для создания списков используются теги ul, ol, dl.
Для определения элементов списка внутри списков ul и ol используется тег li.
Для списка типа dl исполюзуются теги dt - термин и dd - определение терминов.





4.Блоки

Для создания блока используется тег div.
Блоки нужны для заключения совокупности елементов внутри него.
Ширины блока всегда равна ширине страницы или ширине родительского блока, если не задана явно.
Высота блока зависит от содержимого, если не задана явно.
